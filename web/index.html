Что такое Web

Web это мир в котором мы делимся все знаниям книгами это мир в котором есть всё чо изабрёл человек

WEB делиться на дава дела FRONTEND И BACKEND

FRONTEND - клиентская сторона пользовательского интерфейса

BACKEND- Серверная часть WEB

Что такое Single-Page Application И MULTI-Page Application и их отличия

SPA-Single-Page Application - Одностраничное приложение имея всего лишь одну HTML структуру и много js кода который реагирует на перехода на другие станица и нажатие кнопок их рендеринг на ходу плюсы этого способа ненужно перезагружать каждый раз при получении новой информации с сервера, меньше нагрузка на сервер

MPA-MULTI-Page Application- имеет много HTML структур и при каждом получении их с сервера у клиента перегружается страницу и очень большая нагрузка на сервер

итоги

Что новая техналогия SPA работает быстрее и надёжнее для сервера

Запросы на сервер AJAX и JSON

AJAX - это запросы клиента на сервер

JSON - Масив с данными кторый идёт с сервера
